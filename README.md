## About This Project
In this project, I implement Machine Learning algorithms such as Decision Tree (DT), Random Forest (RF), Na√Øve Bayes (NB), K-Nearest Neighbor (KNN), Support Vector Machine (SVM) and Deep Neural Network algorithm in the form of Convolutional Neural Network (CNN) to classify weather from images using the Multi-class Weather Dataset (MWD). The aim of this study is to compare performance between those six algorithms. We use K-Fold Cross Validation with cv = 5 to validate our model. Before predicting the images, we preprocess the images using smoothing and sharpening.
